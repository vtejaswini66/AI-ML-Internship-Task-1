# AI & ML Internship – Task 1

## Task Title
Understanding Dataset & Data Types

## Description
This repository contains analysis of the Titanic Dataset and Students Performance Dataset.
The goal is to understand dataset structure, data types, data quality, and ML readiness.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Files Included
- `task1_analysis.ipynb` – Dataset exploration and analysis
- `dataset_analysis_report.md` – Humanized 1-page analysis report
- `README.md` – Project overview

## Outcome
Clear understanding of dataset structure, data types, missing values, and suitability for machine learning.
